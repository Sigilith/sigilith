# Sigilith

Sigilith is a reproducible, representation-invariant method for structural analysis of symbolic systems.

It operates via:
- canonical segmentation
- behaviour-preserving transformations (BPS)
- metric extraction
- structural signature constraints (SSC)
- classification

## What Sigilith Does

Sigilith classifies symbolic systems structurally without relying on meaning, language, or interpretation.
